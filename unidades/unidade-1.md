# Unidade 1

## Tema

Fundamentos do desenvolvimento Web back-end com NestJS, incluindo autenticação e integração externa.

## Objetivos da unidade

- Compreender arquitetura cliente-servidor e HTTP.
- Configurar o ambiente de desenvolvimento.
- Estruturar uma API básica em NestJS.
- Validar dados e tratar erros.
- Implementar autenticação e autorização básicas.
- Consumir APIs externas e organizar configuração.

## Encontros

- 1 a 20

## Produtos esperados

- API inicial funcionando.
- CRUD simples sem banco.
- Documentação básica com Swagger.
- Rotas protegidas com autenticação.
- Avaliação prática da unidade.
