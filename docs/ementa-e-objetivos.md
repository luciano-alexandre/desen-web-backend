# Ementa e Objetivos

## Ementa

Fundamentos do desenvolvimento Web Back-end e arquiteturas cliente-servidor.
Construção de aplicações em ambiente de framework, tratamento de dados,
formulários, sessões e autenticação. Acesso a APIs e integração com serviços
externos. Operações básicas com bases de dados relacionais e não relacionais e
implantação de aplicações Web.

## Objetivos

- Compreender fundamentos do desenvolvimento Web back-end.
- Utilizar um framework back-end para tratar requisições, validar dados e
  autenticar usuários.
- Realizar deploy de aplicações Web e consumir APIs RESTful.
- Operar bases de dados relacionais e não relacionais.

## Bases científico-tecnológicas (conteúdos)

1. Revisão de front-end e introdução ao desenvolvimento Web back-end.
   1. Comparação de aplicações cliente e servidor.
   2. Modelo de requisição e resposta com HTTP e seus principais métodos.
   3. Visão geral do estado da arte de tecnologias e frameworks.
   4. Arquitetura de aplicações Web back-end.
2. Desenvolvimento de aplicações em framework Web back-end.
   1. Configuração do ambiente de desenvolvimento.
   2. Obtenção e tratamento de informações da aplicação cliente.
   3. Criação e validação de formulários.
   4. Armazenamento de dados temporários em sessão.
   5. Autenticação de usuário.
3. Deploy de aplicações Web.
4. Acesso a APIs RESTful.
5. Acesso a dados.
   1. Operações básicas com bases de dados relacionais.
   2. Operações básicas com bases de dados não relacionais (NoSQL).
