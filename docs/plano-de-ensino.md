# Plano de Ensino

## Identificação

- Disciplina: Desenvolvimento Web Backend
- Curso: Tecnologia em Sistemas para Internet (TSI)
- Carga horária: 60h
- Semestre letivo: graduação
- Framework base: NestJS

## Objetivo geral

Desenvolver aplicações Web back-end em arquitetura cliente-servidor, com uso de
framework, autenticação, integração com APIs, persistência de dados e deploy.

## Metodologia

- Aulas expositivas dialogadas com demonstração de código.
- Laboratórios guiados com checkpoints.
- Estudos dirigidos e leituras curtas.
- Desenvolvimento incremental de projeto integrador.
- Revisões técnicas e apresentações parciais.

## Recursos necessários

- Node.js LTS
- VS Code ou editor equivalente
- Nest CLI
- Insomnia ou Postman
- Docker Desktop ou Podman
- PostgreSQL
- MongoDB
- Git e GitHub
- Plataforma de deploy em nuvem

## Organização em unidades

- Unidade 1: Fundamentos de backend com NestJS e construção de APIs básicas.
- Unidade 2: Formulários, autenticação, sessões, integração externa e testes.
- Unidade 3: Persistência relacional/NoSQL, deploy e projeto final.

## Avaliação

- Unidade 1: exercícios, prática guiada e avaliação prática.
- Unidade 2: checkpoints de autenticação/integração e avaliação prática.
- Unidade 3: projeto final, deploy e apresentação técnica.

## Bibliografia de referência

- Documentação oficial do NestJS.
- Documentação oficial do Node.js.
- Documentações oficiais de PostgreSQL e MongoDB.
- Materiais complementares indicados em `docs/referencias.md`.
