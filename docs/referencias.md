# Referências e Recursos

## Documentação principal

- NestJS: arquitetura, controllers, providers, pipes, guards e interceptors.
- Node.js: runtime, variáveis de ambiente e sistema de módulos.
- TypeScript: tipagem, classes, decorators e configuração.

## Persistência e dados

- PostgreSQL: modelagem relacional, SQL básico e constraints.
- Prisma ou TypeORM: mapeamento de dados e migrações.
- MongoDB e Mongoose: documentos, coleções e operações básicas.

## API e autenticação

- HTTP Semantics e códigos de status.
- JSON Web Token (JWT).
- Passport para estratégias de autenticação.
- OpenAPI/Swagger para documentação.

## Deploy e operação

- Docker e Docker Compose.
- Plataformas de deploy para APIs Node.js.
- Boas práticas de variáveis de ambiente e logs.
